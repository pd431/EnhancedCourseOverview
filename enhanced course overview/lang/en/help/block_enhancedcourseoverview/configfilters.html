Course Filters

This setting configures the filter buttons that will appear above the course list.

The syntax for defining filters is as follows:

  • Empty lines are ignored
  • Lines without a pipe character (|) are treated as group headers
  • Lines with a pipe character have the format: Filter Title|text_to_match

Example:

2023-24
Term 1|_A1_202324
Term 2|_A2_202324

2024-25
Term 1|_A1_202425
Term 2|_A2_202425

This will create two button groups:

  • Group "2023-24" with buttons "Term 1" and "Term 2"
  • Group "2024-25" with buttons "Term 1" and "Term 2"

When a filter button is clicked, only courses containing the specified text in their name or description will be shown.
